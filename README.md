# Crop Recommendation System 🌱

🔗 Live Demo: https://crop-recommendation-j9dj6zappekhbawya3fkmcw.streamlit.app/

This machine learning application recommends the most suitable crop based on:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

### Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
